What is Open GIS?

Open GIS is a free, Open-Source Geographic Information System (GIS).

With no technical knowledge required, users can create, edit, and share Meaningful Maps.

You can use the Open GIS Editor at ogis.app.

The Editor

The Editor runs in your web browser and allows you to:

  • Import data from various formats (GPX, KML and GeoJSON)
  • Create Overlays (Markers, Line & Shapes) using the intuitive interface
  • Add meaning to your Maps by styling Overlays using Types
  • Export your Maps to save them offline, including your custom Types.

The Editor is built with Waymark JS, an Open GIS project for creating and editing Maps and uses OpenStreetMap as the Basemap.

No account or API key is required to use the Editor and Maps are stored in your browser's local storage.

Project Goals

  1. Provide free, Open-Source GIS tools that are easy to use and accessible to everyone.
  2. Create revenue through Pro features and donations to support the project. Pro features include:
    • Cloud storage for Maps
    • Overlay Image hosting
    • Collaboration tools

Progress

The Open GIS project is in its early stages, with the Editor being the first major component. The Editor is fully functional and ready for use.

The next stage is adding accounts and cloud storage for Maps.
